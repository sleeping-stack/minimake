- strchr 是C标准库中的一个字符串处理函数，用于在字符串中查找指定字符的第一次出现位置。

~~~c
#include <string.h>
char *strchr(const char *str, int c);
// 参数说明
// str: 要搜索的字符串（const char *类型）
// c: 要查找的字符（int类型，实际使用时会转换为char）
// 返回值
// 成功: 返回指向第一次出现字符c的指针
// 失败: 返回NULL（未找到指定字符）
~~~

- fgets()函数的工作原理：

从文件中读取一行（遇到'\n'或EOF停止）
将读取的内容（包括换行符'\n'）存储到字符数组中
自动在末尾添加空字符'\0'
